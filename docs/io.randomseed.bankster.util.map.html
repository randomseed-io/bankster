map-keys

added in 1.0.0

(map-keys f m)(map-keys f m dst)

For each key and value of the given map m calls a function passed as the first argument (passing successive keys during calls to it) and generates a map with keys updated by results returned by the function. When a third argument is given, it should be a map on which operations are performed instead of using an empty map.

map-keys-and-vals

added in 1.0.0

(map-keys-and-vals f m)(map-keys-and-vals f m dst)

For each key and value of the given map m calls a function passed as the first argument (passing successive keys during calls to it) and generates a map with keys updated by results returned by the function and values also updated by results of the same function. The function should return a sequential collection of 2 elements: first containing a new value of a key and second containing a new value of a transformed value associated with that key. When a third argument is given, it should be a map on which operations are performed instead of using an empty map.
